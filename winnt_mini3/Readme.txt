
Readme.txt
==========
     AMD PCnet-Family Software
     NDIS3 Miniport Driver for Windows NT 3.51 and 4.0
     Version 4.04


Features Added
==============
This version of the driver has added support for the EXTPHY keyword. 
With the use of the EXTPHY keyword, one may set the speed and duplex mode 
of the controller. 


Controllers Supported
=====================
This driver supports the PCnet Family controllers 
including the PCnet-Fast III controller.
